This type needs to be both printable and sortable

March 2026 · Part A: The Challenge

A leaderboard entry that needs to be displayed and ranked.

struct Score {
    name: String,
    points: u32,
}

fn main() {
    let scores = vec![
        Score { name: "Alice".into(), points: 150 },
        Score { name: "Bob".into(), points: 230 },
        Score { name: "Carol".into(), points: 180 },
    ];

    for s in &scores {
        println!("{}: {}", s.name, s.points);
    }
}

Make Score sortable (so you can call .sort() on a vec of them) and printable with println!("{}", score).

Your options

  • Implement fmt::Display for printing and derive Ord/PartialOrd/Eq/PartialEq for sorting
  • Implement ToString directly and write a custom compare method
  • Derive Debug and use it for both display and sorting via the debug string
  • Use a single to_comparable_string method that handles both concerns

Pick one before reading the solution.
