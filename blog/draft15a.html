This iterator needs to yield borrowed data. Make it compile

March 2026 · Part A: The Challenge

A word list that should be iterable. The iterator should yield references to the words, not copies.

struct WordList {
    words: Vec<String>,
}

impl WordList {
    fn new(words: Vec<String>) -> Self {
        WordList { words }
    }
}

fn main() {
    let list = WordList::new(vec![
        "hello".into(),
        "world".into(),
        "rust".into(),
    ]);

    for word in &list.words {
        println!("{}", word);
    }
}

Write a custom iterator struct for WordList that yields &str references to each word.

Your options

  • Create an iterator struct with a lifetime parameter that borrows the word list, yielding &'a str
  • Have the iterator clone each String into a new String and yield owned values
  • Return &'static str from the iterator by leaking the strings
  • Store the index in the iterator and return &str without any lifetime parameter on the struct

Pick one before reading the solution.
