Find the process using the most memory

March 2026 · Part A: The Challenge

A monitoring tool that finds the heaviest process by memory usage.

struct Process {
    name: String,
    memory_mb: u64,
}

fn main() {
    let procs = vec![
        Process { name: "nginx".into(), memory_mb: 120 },
        Process { name: "postgres".into(), memory_mb: 450 },
        Process { name: "redis".into(), memory_mb: 85 },
    ];

    let mut max_mem = 0;
    let mut heaviest = &procs[0];
    for p in &procs {
        if p.memory_mb > max_mem {
            max_mem = p.memory_mb;
            heaviest = p;
        }
    }

    println!("{}: {} MB", heaviest.name, heaviest.memory_mb);
}

The manual max-tracking loop is verbose. There's an iterator adapter that finds the maximum element by an extracted field. Replace the loop.

Your options

  • Use max_by_key — find the max by an extracted field
  • Use max_by with a full comparator closure
  • Use max() — natural ordering handles it
  • Use sort_by_key then take the last element

Pick one before reading the solution.
