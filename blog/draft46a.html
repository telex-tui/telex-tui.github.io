_while

Read lines from a stream. Stop at the first blank line

March 2026 · Part A: The Challenge

An HTTP-style parser that reads header lines until it hits a blank line separator.

fn main() {
    let lines = vec!["Host: localhost", "Port: 8080", "Accept: */*", "", "body here", "more body"];

    let mut headers = Vec::new();
    for line in &lines {
        if line.is_empty() {
            break;
        }
        headers.push(*line);
    }

    println!("{:?}", headers);
}

The for loop with break is a manual version of a standard iterator adapter. Replace the loop with a single iterator chain.

Your options

  • Use take_while — take elements while the predicate holds, stop at first failure
  • Use filter — skip blank lines and keep the rest
  • Use take(n) — take a fixed number of header lines
  • Use skip_while — skip lines while they're non-empty

Pick one before reading the solution.
