This struct holds a reference. Now give it a method

March 2026 · Part A: The Challenge

A parser that holds a reference to its input.

struct Parser<'a> {
    input: &'a str,
    pos: usize,
}

impl<'a> Parser<'a> {
    fn new(input: &'a str) -> Self {
        Parser { input, pos: 0 }
    }
}

fn main() {
    let source = String::from("hello world");
    let parser = Parser::new(&source);
    println!("parsing: {} (at pos {})", parser.input, parser.pos);
}

Add a next_word method that returns the next whitespace-delimited word as &str.

Your options

  • Write fn next_word(&mut self) -> Option<&str> — lifetime elision handles the return type
  • Explicitly annotate as fn next_word(&'a mut self) -> Option<&'a str> to tie return to input lifetime
  • Remove the lifetime from the struct and clone the input string instead
  • Add a second lifetime parameter for the method's borrow of self

Pick one before reading the solution.
