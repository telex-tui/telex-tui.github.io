Add a parent pointer to this tree node

March 2026 · Part A: The Challenge

A basic tree. Each node owns its children.

struct Node {
    value: String,
    children: Vec<Node>,
}

fn main() {
    let tree = Node {
        value: "root".into(),
        children: vec![
            Node {
                value: "child-1".into(),
                children: vec![],
            },
            Node {
                value: "child-2".into(),
                children: vec![],
            },
        ],
    };

    println!("root: {}", tree.value);
}

Add a parent pointer so each child can navigate back to its parent.

Your options

  • Store nodes in a Vec<Node> arena and use usize indices as parent/child pointers
  • Add a parent: Box<Node> field so each child owns a copy of its parent
  • Add a parent: &Node reference field with a lifetime parameter on the struct
  • Wrap every node in Rc<RefCell<Node>> and store the parent as the same type

Pick one before reading the solution.
